# Usage

- CLI commands: `--help`, `--version`, `--daemon`, `--cli`
- Profile:
	- `--profile-show`
	- `--profile-set-name "Name"`
	- `--profile-set-bio "Bio"`
	- `--profile-set-status active|hidden|inactive`
- Messaging (CLI):
	- `--send --chat <id> --from <user> --to <user> --message "text"`
	- `--edit --id <message_id> --message "new text"`
	- `--read --id <message_id>`
	- `--history --chat <id> --limit <n>`
- Peer discovery:
	- `--discover --discover-timeout 1200`
	- `--peers`
	- `--peer-add host:port:name`
- Local paths/cache:
	- `--paths`
	- `--clear-cache`
- Messenger TUI: `--tui`
- API:
	- `--api-info` (prints `api_endpoint` and token)
	- Localhost JSON API: first `auth.login`, then:
		- `system.ping`
		- `profile.get`, `profile.update`
		- `message.send`, `message.edit`, `message.read`, `message.history`
		- `peers.list`, `peers.discover`, `peers.add`
